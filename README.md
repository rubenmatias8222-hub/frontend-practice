## 🌐 Deployment Options

### Option 1: GitHub Pages (Free & Recommended)
1. Create a GitHub account at github.com
2. Create a new repository named `portfolio`
3. Upload all files to the repository
4. Go to Settings > Pages
5. Select "Deploy from branch" and choose main branch
6. Your site will be live at `yourusername.github.io/portfolio`

### Option 2: Netlify (Free with advanced features)
1. Sign up at netlify.com
2. Drag and drop the entire folder or connect your GitHub repo
3. Netlify will auto-deploy with a free domain
4. Get instant previews on every push

### Option 3: Vercel (Free)
1. Sign up at vercel.com
2. Import your GitHub repository
3. Click "Deploy"
4. Automatically deployed with zero config

### Option 4: Firebase Hosting (Free tier)
1. Install Firebase CLI: `npm install -g firebase-tools`
2. Initialize: `firebase init`
3. Deploy: `firebase deploy`

## Customization Guide

### Update Social Media Links
In `index.html`, find the social-icons section and update URLs:
```html
<a href="https://linkedin.com/in/yourprofile" target="_blank">
```

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #3498db;
    --accent-color: #e74c3c;
    --secondary-color: #2c3e50;
}
```

### Add More Projects
Copy the project-card section and update with your own projects:
```html
<article class="project-card">
    <div class="project-image" style="background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);"></div>
    <h3>Your Project Title</h3>
    <p>Your project description</p>
    <div class="project-tags">
        <span>Technology</span>
    </div>
    <a href="#" class="btn btn-secondary">View Project</a>
</article>
```
